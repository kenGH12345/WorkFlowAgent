# CODEX FORGE — Media Assets

## Social Preview (1280×640)

1. Open `social-preview.html` in Chrome
2. Open DevTools → set device to 1280×640
3. Screenshot the full page
4. Upload to: GitHub repo → Settings → General → Social preview

## Demo GIF

Record a ~30 second demo showing:
1. User types a one-sentence requirement
2. Analyst agent processes the requirement
3. Architect generates the design
4. Developer writes the code
5. Tester validates the output

### Recommended recording tools:
- **Windows**: [ScreenToGif](https://www.screentogif.com/) or OBS + gifski
- **Terminal**: [VHS](https://github.com/charmbracelet/vhs) by Charmbracelet
- **Cross-platform**: [asciinema](https://asciinema.org/) for terminal recordings

### Specs:
- Resolution: 800×450 (2x for retina)
- Duration: < 30 seconds
- File size: < 5MB for fast README loading
- Format: GIF or MP4 (GitHub supports both)

Save the output as `demo.gif` in this directory.
